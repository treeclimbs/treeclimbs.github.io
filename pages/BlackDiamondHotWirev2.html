Features:the original climbing wiregate
Description:

Although wiregate carabiners were common in the marine industry, the BD HotWire was the first purpose built climbing carabiner using this technology. The design had fewer moving parts, and testing revealed that the lower mass of the gate resulted in less succeptibility to gate-flutter than solid gate carabiners.

The BD QuickWire was a lower cost model using the Quicksilver body.

The BD website from 2002-2004 shows a HotWire with the classic Chouinard C-diamond logo. This example has the more modern BD double-diamond logo, so I have categorized this as v2 due to this marking. The body appears to be the same shape. I aquired this example around 2006.
