Description & Commentary:

This appears to be a common accessory carabiner used for promotion or gear storage, but it has a few features not commonly found on accessory carabiner:

  • a rollpin for the nose pin, but a riveted pin for the hinge
  • hooked notch, to actually catch the nose pin
  • gate features a full interior shield

Stamped to show an EMS (Eastern Mountain Sports) logo.

Reverse spine is stamped (rollstamped?) "NO LOAD BEARING"

The shape and features is almost identical to an Omega accessory carabiner (2024.60.012), which suggests manufacture by Omega as EMS did not make their own carabiners. See also: CMC Accessory Carabiner.
