Description & Commentary:

YOKE N-256G v1 and v2 differ in the following minor ways:

CE #

Size of markings on gate, Other markings very slightly different in size

In addition to the Technical Notice, YOKE also provides a general Warning Section: Connecting Components.

This carabiner typifies a basic industrial use steel carabiner. Steel construction provides tremendous strength and durability (and useful throwing weight on the end of a lanyard). The gate meets modern ANSI gate (plus minor axis and side load) strength requirements. It has been drilled for the installation of a captive pin to help prevent crossloading.

In contrast with the triple-action (double autolocking) carabiners used in tree climbing work, this industrial work carabiner only requires two actions (single twist-lock) to open.
